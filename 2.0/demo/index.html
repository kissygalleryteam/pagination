仿淘宝皮肤，自定义参数调用

new Pagination('demo-container-2', {
currentPage: 7, // 默认选中第7页
totalPage: 12, // 一共有12页
firstPagesCount: 2, // 显示最前面的2页
preposePagesCount: 1, // 当前页的紧邻前置页为1页
postposePagesCount: 2, // 当前页的紧邻后置页为2页
lastPagesCount: 1 // 显示最后面的1页
});

请点击页码
